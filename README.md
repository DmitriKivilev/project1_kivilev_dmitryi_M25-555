# Лабиринт Сокровищ

Текстовая игра на Python.Жанр - приключение. Игрок продивигается по лабиринту исследуя комнаты, собирает предметы, решает загадки и ищет сокровище.

## Установка

```bash
# Установка зависимостей
make install
# или
poetry install
```

## Запуск игры

```bash
# Запуск 
make project
# или
poetry run## Демонстрация игрового процесса
